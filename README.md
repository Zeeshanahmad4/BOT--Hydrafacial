## Getting Started

### Prerequisites
* beautifulsoup
```sh
pip install bs4
```


### Installation
1. Clone the repo
```sh
git clone https://github.com/Zeeshanahmad4/BOT--Hydrafacial.git
```
2. Install python 
```sh
install python
```
3. Install python packages
```sh
pip install bs4
```
